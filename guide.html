簡介

Minecraft 貼圖大師 v0.9.1 是一個專為 Minecraft 材質包設計的像素藝術編輯器，支援多分頁編輯、像素級精細操作、圖像變換、主題切換等功能。本手冊將詳細介紹如何使用此工具，幫助您快速創建或編輯 Minecraft 材質貼圖。

1. 介面介紹

啟動程式後，您將看到以下主介面：

  • 左上角功能列：提供檔案操作、編輯、圖像變換和設定選單。
  • 左側邊欄：包含編輯工具和顏色調整功能。
  • 右側編輯區：顯示當前開啟的圖像（若無開啟檔案則為空白）。
  • 右下角儲存按鈕：快速儲存當前圖像。
  • 右上角主題切換按鈕：月亮圖示（🌙）用於切換亮/暗主題。
    • 注意：暗色主題目前不穩定，建議使用預設亮色主題。
  • 底部狀態列：顯示操作資訊（如縮放比例、檔案狀態等）。

2. 檔案操作

2.1 開啟檔案

  1. 點擊功能列「檔案 > 開啟檔案」或使用快捷鍵 Ctrl + O。
  2. 選擇 PNG 格式檔案（目前僅支援 PNG）。
  3. 若檔案尺寸非 16×16（Minecraft 標準貼圖尺寸），會顯示以下提示：

    • 是：將圖像縮放至 16×16，適合快速生成貼圖雛型。例如：

      - 縮放後（16×16）：
    • 否：保留原始尺寸，適合生物貼圖等非標準尺寸。
    • 取消：取消開啟操作。

2.2 建立新圖片

  1. 點擊功能列「檔案 > 建立圖片」或使用快捷鍵 Ctrl + N。
  2. 彈出對話框，提供預設尺寸選項：
    • Minecraft 物品/方塊 (16×16)：標準物品或方塊貼圖。
    • Minecraft 效果圖示 (18×18)：效果圖示貼圖。
    • Minecraft 貼圖動畫 (16×16n)：動畫貼圖，需指定幀數。
    • 自訂：輸入自訂寬度和高度。
  3. 點擊「確定」後，將在新分頁創建透明圖像。

點擊功能列「檔案 > 快速載入範例」。載入一個 16×16 的範例圖像，適合練習操作：

2.4 儲存檔案

點擊右下角「儲存圖片」按鈕或使用快捷鍵 Ctrl + S。若檔案已有路徑，則直接儲存；否則，彈出檔案選擇對話框，預設檔名為分頁標題（去掉「*」）。儲存格式僅支援 PNG。

2.5 關閉分頁

點擊功能列「檔案 > 關閉分頁」或使用快捷鍵 Ctrl + W。若分頁有未儲存變更，會提示是否儲存。

2.6 儲存所有分頁

點擊功能列「檔案 > 儲存所有」或使用快捷鍵 Ctrl + Shift + S，儲存所有有變更的分頁。

3. 編輯操作

3.1 縮放

在編輯區使用滑鼠滾輪進行縮放：向上滾動放大（最大 5000%），向下滾動縮小（最小 1%）。縮放比例顯示於底部狀態列。

3.2 編輯工具

左側邊欄提供以下編輯工具：

3.2.1 畫筆

  • 功能：繪製像素或線條。
  • 顏色：點擊「選擇顏色」按鈕選擇顏色，選定後按鈕邊框顯示當前顏色：
  • 粗細：使用「畫筆粗細」滑桿調整（1 到 10）。
  • 預覽：勾選「顯示畫筆預覽圓圈」顯示滑鼠位置的虛線圓。

3.2.2 滴管

  • 功能：選取圖像中某像素的顏色，選定後自動切回畫筆工具。

3.2.3 橡皮擦

  • 功能：擦除像素（設為透明），粗細由「畫筆粗細」滑桿控制。

3.2.4 填滿

  • 功能：將點擊處及相鄰的相似顏色像素填充為選定顏色。
  • 容差值：使用「容差值」滑桿（0 到 255）調整顏色相似度。

3.2.5 移動

  • 功能：移動選取區域的內容。

3.2.6 複製與貼上

  • 複製：選擇「複製」或使用 Ctrl + C。
  • 貼上：選擇「貼上」或使用 Ctrl + V，顯示貼上預覽：
  • 操作：移動滑鼠調整位置，左鍵點擊、按 Enter 確認，按 ESC 取消。

3.2.7 圖像變換

  • 提供水平翻轉、垂直翻轉、順時針/逆時針旋轉 90°、以及重置編輯功能。

3.2.8 復原與重做

  • 復原：點擊「復原」或使用 Ctrl + Z。
  • 重做：點擊「重做」或使用 Ctrl + Y。

4.1 像素選擇器

5. 注意事項

  • Beta 版限制：程式為 Beta 版，可能存在錯誤或不穩定情況。
  • 圖像尺寸：Minecraft 材質包通常為 16×16，建議優先使用此尺寸。
  • 檔案格式：目前僅支援 PNG 格式。
  • 效能：對於大型圖像或頻繁操作，可能出現延遲，建議儲存後重新開啟。
