簡介

「Minecraft 貼圖大師」是一款輕巧且強大的像素藝術編輯器，專為 Minecraft 材質包創作者量身打造。無論您是想修改現有材質，還是從零開始創作全新的像素藝術，本程式都能提供您所需的工具，讓創作變得更加簡單有趣。

立即下載

重要提醒

系統需求：本程式僅支援 Windows 系統，且需要安裝 Java 環境（JRE/JDK）才能正常運行。

版本狀態：目前為 Beta 測試版，可能存在不穩定情況，建議定期儲存作品。

主要功能

🎨

像素級編輯

專為 16×16 像素材質設計，支援畫筆、橡皮擦、滴管、填滿等工具

📑

多分頁編輯

同時開啟多個材質檔案，提升工作效率

🔄

圖像變換

支援翻轉、旋轉、顏色調整等多種變換操作

🎯

精確選取

像素選擇器提供 32 倍放大檢視，精確編輯每個像素

🌙

主題切換

支援亮色與暗色主題（暗色主題測試中）

📏

多種尺寸

支援 16×16 物品、18×18 效果圖示、動畫材質等
